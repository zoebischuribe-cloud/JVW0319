## 🎯 核心要求（必须遵守）

### 1. 使用最新数据 🔴

**要求**:
- ✅ **始终使用最新可用的真实数据**
- ✅ **定期检查数据更新**（每次研究前）
- ✅ **优先选择最近 2-3 年的数据**
- ❌ **禁止使用超过 10 年的旧数据**（除非特殊原因）

**检查清单**:
```
□ 数据年份：是否最新？
□ 数据周期：是否最新可用？
□ 数据来源：是否官方？
□ 样本量：是否充足？
□ 检测方法：是否先进？
```

**示例**:
- ✅ 正确：NHANES 2021-2022（最新）
- ❌ 错误：NHANES 2005-2006（18 年前）

---

### 2. 必须使用真实数据 🔴

**要求**:
- ✅ **100% 使用真实数据**
- ✅ **数据来源可追溯**
- ✅ **数据质量可验证**
- ❌ **绝对禁止使用模拟数据**
- ❌ **绝对禁止伪造数据**
- ❌ **绝对禁止篡改数据**

**验证清单**:
```
□ 数据来源：官方数据库/公开数据集
□ 数据位置：本地已下载
□ 数据验证：样本量、变量、统计
□ 数据引用：正确的引用格式
□ 数据质量：完整率、准确性
```

**数据来源优先级**:
1. ✅ **官方数据库**（CDC、NCBI、WHO 等）
2. ✅ **公开数据集**（Kaggle、PhysioNet 等）
3. ✅ **同行评审研究数据**
4. ❌ **模拟数据**（禁止）
5. ❌ **伪造数据**（禁止）

---

### 3. 数据时效性检查 🔴

**每次研究前必须执行**:

```python
# 伪代码示例
def check_data_freshness(data_source):
    latest_year = get_latest_available_year(data_source)
    current_year = 2026
    
    if current_year - latest_year > 10:
        raise Warning("数据超过 10 年，请检查是否有更新版本")
    
    if current_year - latest_year > 5:
        print("⚠️  数据已超过 5 年，建议寻找更新数据")
    
    return latest_year
```

**检查频率**:
- ✅ 每次开始新研究时
- ✅ 每次数据分析前
- ✅ 每次论文撰写前
- ✅ 收到审稿意见后（检查是否有更新数据）

## 🔍 数据验证流程

### 步骤 1: 来源验证

```
□ 数据来源是否官方/权威？
□ 是否有同行评审支持？
□ 是否被广泛引用？
□ 是否有明确的数据采集方法？
```

### 步骤 2: 时效性验证

```
□ 数据年份？
□ 是否最新可用？
□ 是否有更新版本？
□ 数据收集时间范围？
```

### 步骤 3: 质量验证

```
□ 样本量是否充足？
□ 完整率如何？
□ 是否有缺失值处理？
□ 是否有质量控制措施？
```

### 步骤 4: 可重复性验证

```
□ 数据是否公开可获取？
□ 是否有详细文档？
□ 是否有代码/脚本支持？
□ 其他研究者能否复现？
```
